Com a conversão das imagens DICOM para jpeg o modo de visualização não se torna um
empecilho para o sistema. As conversões realizadas em um conjunto com a biblioteca
Java Advanced Imaging e a biblioteca para gerenciamento do DICOM deixaram imagens
de formato dcm de 15mb com aproximamente 210kb. A biblioteca desenvolvido para a
visualização das imagens médicas através do navegador trabalha bem com as imagens
jpeg, exigindo um pequeno tempo para manipulação dessas imagens de 210kb.

Um grande cuidado deve ser tomado com o sistema de arquivos.
